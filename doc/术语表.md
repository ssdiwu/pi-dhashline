# 术语表

## DHashline

本项目提供的文件级内容标签与行号锚定编辑协议及其 Pi 工具实现。

_Avoid_：把 DHashline 泛指为任意逐行哈希编辑器。

## 文件标签

完整规范化文件内容对应的 8 位大写十六进制标识，写作 `[PATH#TAG]` 中的 `TAG`。

_Avoid_：行哈希、版本号。

## 文件段

一条 `[PATH#TAG]` header（头部）及其后续编辑操作组成的单文件补丁单元。

## 锚点

文件标签签发时视图中的 1-based（从 1 开始）行号；标签绑定文件状态，行号定位该状态中的内容。

## 过期恢复

当前文件已偏离标签快照时，仅在目标内容未变、映射唯一且所有锚点偏移一致的条件下，把编辑重映射到当前文件的过程。

_Avoid_：模糊重定位、猜测修复。

## 已见行

同一文件标签下，模型通过 `read` 或 `search` 实际收到的源文件行号集合。

## 人类可读投影

公开工具在 Pi UI 中对机器结果的两层文字呈现：默认只显示紧凑动作摘要，按 `Ctrl+O` 展开完整锚点、编辑动作、diff 或错误指导；两层都不直接展示原始 JSON 或内部 details。
